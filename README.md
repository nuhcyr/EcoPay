# EcoPay
EcoPay; kurumsal finans yönetimi ile sürdürülebilirlik hedeflerini birleştiren, Yapay Zeka (AI) destekli bir "Yeşil Fintech" platformudur. Sistem, çalışanlara tanımlanan sanal kartlar üzerinden yapılan harcamaları gerçek zamanlı olarak analiz ederek, çevre dostu tüketimi ödüllendirmeyi ve şirketlerin karbon ayak izini dijital olarak yönetmesini amaçlar.
